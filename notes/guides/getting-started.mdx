Welcome to **Grimoire** — notes written in MDX, rendered as a fast little app,
and shippable as a single binary.

<Callout type="tip" title="No imports needed">
Built-in components like `<Chart/>`, `<DataTable/>` and `<Callout/>` are always
available. Drop a component in `components/` and it's auto-registered too.
</Callout>

## A chart

<Chart
  type="bar"
  title="Quarterly revenue (k$)"
  data={{
    labels: ["Q1", "Q2", "Q3", "Q4"],
    datasets: [
      { label: "2025", data: [42, 55, 61, 70] },
      { label: "2026", data: [50, 64, 73, 88] },
    ],
  }}
/>
