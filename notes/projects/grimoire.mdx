You're reading a note rendered by the very system it describes. **Grimoire** turns a
folder of MDX files into a fast, searchable little app — and then folds that whole app,
runtime included, into a single self-contained binary you can copy onto any machine.

## The pitch

Write Markdown. When prose isn't enough, drop in a component — a `<Chart/>`, a
`<DataTable/>`, a `<Quiz/>` — no imports, no wiring. The build statically renders every
note, bundles the interactive islands, and `bun build --compile` stamps it all into one
executable. No Node install, no `npm i` on the target box, no Docker. Just `./grimoire`.

## How it's wired

A note is just a file. `gray-matter` peels off the YAML frontmatter, `@mdx-js` compiles
the body, and `remark-gfm` + `rehype-slug` + `rehype-highlight` handle tables, anchors and
syntax highlighting. Preact renders the tree; Tailwind v4 paints it; Chart.js draws the
graphs. Bun ties the whole thing together at build time.

```ts
// the shape every note resolves to
type Note = {
  slug: string;          // "projects/grimoire"
  frontmatter: Frontmatter;
  Component: FunctionComponent; // the compiled MDX
  headings: Heading[];   // pulled for the on-page TOC
};
```
